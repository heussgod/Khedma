# Khedma

Khedma is a trust-first services platform prototype built for a capstone AI design project.

The product is focused on Morocco and starts with vetted at-home beauty services in Casablanca. It is designed as a two-sided platform:

- customers can discover and book trusted local professionals
- workers can apply, verify their profile, and build reputation over time

The prototype was built in SwiftUI and explores how AI-informed product systems can improve trust, matching, reputation, support, and service coordination in a local marketplace.

## Stack

- Swift
- SwiftUI
- Xcode
- local mock data only

## Current product scope

- bilingual onboarding
- customer and worker flows
- verified provider discovery
- service map and neighborhood presence
- booking flow
- favorites, support, and profile screens

## Project goal

Khedma is designed to show a premium, localized, operator-backed services experience that feels more trustworthy than a generic gig marketplace.
